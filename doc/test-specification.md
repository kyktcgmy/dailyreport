# テスト仕様書 - 営業日報システム

## テスト方針

| 種別 | 目的 | 対象 |
|---|---|---|
| APIテスト | エンドポイントの正常系・異常系・権限制御を検証 | 全APIエンドポイント |
| 画面テスト（E2E） | ユーザー操作フロー・バリデーション・表示制御を検証 | 全画面 |

### テストユーザー定義

| ユーザー名 | ロール | 備考 |

| 山田 太郎（yamada@example.com） | sales | 上長：上長 花子 |
| 佐藤 次郎（sato@example.com） | sales | 上長：上長 花子 |
| 上長 花子（hanako@example.com） | manager | 部下：山田・佐藤 |
